********************************************************************
Sieve of Eratosthenes program: a simple, ancient algorithm for
finding all prime numbers up to any given limit.

Author: Anugesh Balasubramaniam
Date: 03/07/2018
********************************************************************

C Program
   -   gcc -Wall -lm sieve.c -o sieve
   -   ./sieve

Fortran Program
   -   gfortran sieve.f95 -o sieve
   -   ./sieve

Python Program
   -   python sieve.py

Ada Program
   -   gnatmake sieve.adb
   -   ./sieve

Cobol Program
   -   cobc -x -free -Wall sieve.cob calcPrime.cob
   -   ./sieve


***NOTE***

The user can only enter a 7 digit number for their upper limit (highly recommend lower than or equal to 1000000) in all five of my programs.
